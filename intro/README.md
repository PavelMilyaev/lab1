# Лабораторная работа №1 - Алгоритмы на Python

## Структура проекта
- `main.py` - главный модуль для запуска всех демонстраций
- `00_distance.py` - расстояния между городами
- `01_circle.py` - площадь круга и проверка точек
- `02_operations.py` - математические операции
- `03_favorite_movies.py` - работа со строками
- `04_my_family.py` - данные о семье
- `05_zoo.py` - управление зоопарком
- `06_songs_list.py` - время звучания песен
- `07_secret.py` - декодирование сообщения
- `08_garden.py` - анализ множеств
- `09_shopping.py` - поиск минимальных цен
- `10_store.py` - расчет стоимости товаров
- `test_main.py` - тесты для проверки
- `requirements.txt` - зависимости

## Запуск

### Все демонстрации:
```bash
python main.py
